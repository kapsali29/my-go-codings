# Simple task cli

## Instructions

How to start a project in golang without github

```shell
go mod init simple-task-cli
```
How to run the project

```shell
go run main.go
```

How to run this cli project

```shell
go run .\main.go add -Name "clean your car" -Id 45 -Description "Clean your car today" -store
go run .\main.go get -Id 45
```

How to prettify code in VSCode

```shell
Shift + Option + F
```
